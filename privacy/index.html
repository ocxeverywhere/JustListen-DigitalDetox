1. 수집·처리하는 정보

본 앱은 회원가입, 로그인, 자체 계정 서버를 제공하지 않으며 사용자를 식별할 수 있는 계정 정보 (이름, 이메일, 전화번호 등)를 직접 수집하지 않습니다. 다만 앱 기능 제공을 위해 아래 정보를 기기 내부에서 저장하거나, 사용자가 요청한 기능 수행에 필요한 범위에서 제3자 서비스로 전송할 수 있습니다.

항목 처리 방식 외부 전송 여부
사용자가 입력한 YouTube URL 자막을 가져오고 듣기 세션을 저장하기 위해 로컬 데이터베이스(Room)에 저장 YouTube 페이지와 자막을 요청하기 위해 YouTube/Google로 전송
YouTube 영상 제목, 채널명, 자막 출처 정보 라이브러리 표시와 원본 출처 표시를 위해 기기 내부에 저장 앱 자체 서버로 전송하지 않음
YouTube 자막 및 정리된 텍스트 기기 내부 파일 영역에 저장하여 재생 세션 복구와 재생에 사용 선택적 Gemini 정리 또는 Google Cloud TTS 사용 시 Google로 전송
음성 합성 오디오 파일 및 재생 캐시 앱 전용 내부 저장소의 audio-cache에 WAV 파일과 매니페스트로 저장 전송하지 않음
앱 설정 언어, 자막 정리 방식, TTS 보이스 설정을 암호화된 SharedPreferences에 저장 전송하지 않음

본 앱은 위치 정보, 연락처, 통화 기록, SMS, 카메라, 마이크, 캘린더, 사진/동영상 라이브러리 전체 접근, 신체 활동, 광고 식별자 등 민감 정보나 광고 목적 식별자를 수집하지 않습니다.

2. 앱 권한 사용 목적

본 앱은 Android에서 다음 권한을 사용합니다.

  • 인터넷(android.permission.INTERNET): YouTube 자막과 영상 메타데이터를 가져오고, Google Cloud Text-to-Speech 및 선택적 Gemini 자막 정리 기능을 호출하기 위해 사용합니다.
  • 포그라운드 서비스 및 미디어 재생 포그라운드 서비스: 사용자가 앱을 벗어나도 듣기 세션의 오디오 재생을 안정적으로 유지하기 위해 사용합니다.
  • 알림(android.permission.POST_NOTIFICATIONS): Android 13 이상에서 재생 상태와 미디어 컨트롤 알림을 표시하기 위해 사용합니다.

본 앱은 카메라, 마이크, 위치, 연락처, 저장소 전체 접근 권한을 요청하지 않습니다.

3. 데이터 보관 위치 및 기간

본 앱이 생성한 듣기 세션 데이터는 기본적으로 사용자의 기기 내부에만 보관됩니다.

  • 저장소: 앱 전용 내부 저장 영역의 로컬 데이터베이스(Room), 암호화된 SharedPreferences, 앱 비공개 파일 디렉터리
  • 저장 데이터: YouTube URL, 영상 제목, 채널명, 자막 출처, 정리된 자막 텍스트, TTS 오디오 캐시, 재생 세션 메타데이터
  • 보관 기간: 사용자가 듣기 세션을 삭제하거나 앱 데이터를 삭제하거나 앱을 제거할 때까지

본 앱은 자체 클라우드 동기화, 자체 계정 백업, 자체 서버 업로드 기능을 제공하지 않습니다.

4. 제3자 서비스 및 외부 전송

본 앱은 사용자가 요청한 듣기 기능을 제공하기 위해 아래 제3자 서비스를 사용합니다.

4.1 YouTube 및 Google YouTube 관련 엔드포인트

  • 전송 데이터: 사용자가 입력한 YouTube URL 또는 영상 ID, 자막 요청에 필요한 네트워크 요청 정보
  • 목적: 영상 제목, 채널명, 사용 가능한 자막 트랙, 자막 본문을 가져오기 위함
  • 수신자: Google LLC 및 YouTube 서비스
  • 참고: YouTube 이용에는 YouTube 및 Google의 정책이 적용될 수 있습니다.

4.2 Google Cloud Text-to-Speech

  • 전송 데이터: 사용자가 듣기를 요청한 정리된 자막 텍스트
  • 목적: 텍스트를 음성 오디오로 변환하여 앱에서 재생하기 위함
  • 수신자: Google LLC
  • 참고: Google Cloud Text-to-Speech 데이터 사용 정책 을 따릅니다.

4.3 Google Gemini API

  • 전송 데이터: 사용자가 선택한 설정에 따라 정리 대상 자막 텍스트와 원본 출처 정보
  • 목적: 자막의 반복, 불필요한 말, 인식 오류 등을 줄여 듣기 좋은 텍스트로 다듬기 위함
  • 수신자: Google LLC
  • 참고: Gemini API 약관 을 따릅니다. Gemini 정리 기능은 설정과 API 키 구성에 따라 사용될 수 있으며, 기본 규칙 기반 정리만 사용하는 경우에는 Gemini로 자막 텍스트를 전송하지 않습니다.

본 앱은 자체 서버를 통해 사용자의 YouTube URL, 자막, 오디오 캐시, 앱 설정을 수집하거나 보관하지 않습니다.

5. 데이터 삭제 방법

사용자는 언제든 본 앱이 저장한 데이터를 다음 방법으로 삭제할 수 있습니다.

방법 1: 앱 내에서 듣기 세션 삭제

라이브러리 화면에서 삭제할 듣기 세션의 삭제 버튼을 사용하면 해당 세션의 로컬 데이터베이스 기록, 정리된 자막 파일, 음성 오디오 캐시가 삭제됩니다.

방법 2: Android 설정에서 앱 데이터 전체 삭제

  1. Android 설정에서 앱 또는 애플리케이션 메뉴를 엽니다.
  2. JustListen을 선택합니다.
  3. 저장공간에서 데이터 삭제 또는 저장공간 지우기를 실행합니다.

방법 3: 앱 제거

본 앱을 제거하면 기기에 저장된 본 앱의 로컬 데이터가 함께 삭제됩니다.

본 앱은 외부 자체 서버에 사용자 데이터를 보관하지 않으므로, 자체 서버 데이터 삭제를 위해 개발자에게 별도 요청할 필요가 없습니다. 다만 문의가 있는 경우 아래 연락처로 요청해 주시면 안내해 드립니다.

Privacy Policy (English Summary)

App: JustListen

Package: io.ocxeverywhere.justlisten

Effective: May 25, 2026

JustListen is an Android app that helps users listen to YouTube content by fetching captions, cleaning transcript text, synthesizing speech, and saving listening sessions locally on the device. The app does not provide account registration or collect personal account information such as name, email address, or phone number.

  • YouTube URL and captions: Sent to YouTube/Google only as needed to fetch video metadata and captions.
  • Google Cloud Text-to-Speech: Receives cleaned transcript text that the user asks the app to turn into audio.
  • Google Gemini API: May receive transcript text only when optional Gemini transcript cleanup is enabled/configured.
  • Local storage: Saved sessions, transcript files, TTS audio cache, and app settings are stored on the user's device.
  • No ads: The app does not use advertising SDKs or collect advertising identifiers.
  • Deletion: Delete sessions in the app, clear app data in Android Settings, or uninstall the app.
  • Children: The app is not directed to children under 13 and does not knowingly collect children's personal information.

For questions, contact ocxeverywhere@gmail.com.
